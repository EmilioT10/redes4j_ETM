Redes de Computadoras

Estudio completo sobre comunicación digital

Origen

Las redes de computadoras tienen su origen en la década de 1950, cuando surgió la necesidad de compartir información entre sistemas militares y científicos. Uno de los hitos más importantes fue la creación de ARPANET en 1969, considerada la primera red de computadoras moderna. Durante los años 80 se desarrolló el protocolo TCP/IP, el cual permitió la interconexión de redes a nivel mundial. Finalmente, en 1989, Tim Berners-Lee creó la World Wide Web, lo que facilitó el acceso a la información y dio lugar al internet que conocemos hoy.

Definición

Una red de computadoras es un sistema formado por múltiples dispositivos interconectados entre sí mediante diferentes medios de transmisión. Estos dispositivos pueden compartir recursos como archivos, impresoras, servicios y conexión a internet. Las redes permiten la comunicación eficiente entre usuarios y sistemas, optimizando procesos y facilitando el trabajo colaborativo.

Tipos de Redes

PAN

Red personal que conecta dispositivos cercanos como teléfonos, audífonos o laptops mediante Bluetooth o USB.

LAN

Red de área local utilizada en hogares, escuelas u oficinas para compartir recursos y conexión a internet.

MAN

Red que cubre una ciudad o campus, conectando múltiples redes locales con alta velocidad.

WAN

Red de gran alcance que conecta países o continentes, siendo Internet el ejemplo más importante.

Medios de Conexión

Fibra Óptica

Utiliza luz para transmitir datos a gran velocidad y largas distancias con mínima interferencia.

Par Trenzado

Cable utilizado en redes LAN, económico y fácil de instalar.

Wi-Fi

Tecnología inalámbrica que permite conectar dispositivos sin cables.

Bluetooth

Conexión de corto alcance usada en dispositivos personales.

Dispositivos

Router

Encargado de dirigir el tráfico de datos entre redes diferentes.

Switch

Conecta dispositivos dentro de una red local y envía datos de manera eficiente.

Hub

Dispositivo básico que distribuye datos a todos los equipos.

Módem

Convierte señales digitales y analógicas para la comunicación.

Bridge

Divide una red en segmentos más pequeños para mejorar el rendimiento.

Gateway

Permite la comunicación entre redes con diferentes protocolos.

Modelo OSI

El modelo OSI divide el proceso de comunicación en siete capas, facilitando la interoperabilidad entre sistemas.

Arquitectura

Cliente - Servidor

Estructura: Un servidor central gestiona recursos.

Funcionamiento: Los clientes envían solicitudes y el servidor responde.

Ventajas: Seguridad, control y escalabilidad.

Desventajas: Dependencia del servidor y costo.

Ejemplos: Google, Facebook, servicios web.

P2P

Estructura: Todos los nodos son iguales.

Funcionamiento: Intercambio directo de recursos.

Ventajas: Bajo costo y fácil implementación.

Desventajas: Menor seguridad y control.

Ejemplos: Torrent, redes de intercambio.
