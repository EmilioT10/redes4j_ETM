Conexiones Ethernet

Las conexiones Ethernet permiten comunicar computadoras mediante cables de red (UTP). Existen dos formas comunes de conexión:

🔹 Computadora a Computadora (Cable Cruzado)

Se utiliza un cable cruzado cuando se conectan directamente dos computadoras sin usar un dispositivo intermedio. Este cable intercambia los cables internos para permitir la comunicación.

  • No requiere switch ni router
  • Ideal para compartir archivos
  • Configuración manual de IP

🔹 Computadora a Concentrador (Switch/Hub)

Se utiliza un cable directo para conectar una computadora a un switch o hub. Es el método más utilizado en redes domésticas y escolares.

  • Permite conectar varios equipos
  • Acceso a internet
  • Configuración automática
